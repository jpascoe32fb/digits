Rules:
This is a game of endurance and mathematical prowess. Each puzzle has a target number and 6 individual starting numbers that you must use in mathematical calculations to try and create the target number. Each of the 6 numbers can only be used once but the answer to a given math calculation becomes a new number to use. Once you finish a puzzle, the next button can be used to randomly generate the next puzzle. You obtain more points the more of the puzzles that you solve. The objective is to get as many points as possible.
